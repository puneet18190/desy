playVideoEditorComponent

(
  • component
  • with_scroll
)

Defined in app/assets/javascripts/video_editor.js:1409

Plays a component and handles the transition to the next one (if there is any). In case the component is of type image or text, the time increase is handled by automaticIncreaseVideoEditorPreviewTimer; if the component is of type video the transition is handled by initializeActionOfMediaTimeUpdaterInVideoEditor and the time increase by increaseVideoEditorPreviewTimer.

Parameters:

  • component Object

    the component to be played

  • with_scroll Boolean

    if true, it uses followPreviewComponentsWithHorizontalScrollInVideoEditor to follow the timeline with the scroll.
